## Vision
Скрипт принимает словарь записей `name: amount`, представляющий количественные данные, и отрисовывает для них наглядную табличку в stdout.

Для примера возьмём метрики годового расхода:
```
EACH_MONTH = 365 // 30; EACH_WEEK = 365 // 7; EACH_2_WEEKS = 365 // 14
annual_exp = {
    'dorm': 1_500 * EACH_MONTH,
    'food': 500 * EACH_WEEK,
    'train': 136 * EACH_2_WEEKS,
    'phone': 713 * EACH_MONTH,
    'music': 99 * EACH_MONTH,
    'parties': 1_500 * EACH_MONTH
}
print_table(annual_exp, ' ANNUAL EXPENDITURE ', '₽')
```

Скрипт отобразит их следующим образом:
```
--------------------------------------------------------------
--------------------- ANNUAL EXPENDITURE ---------------------
--------------------------------------------------------------
   dorm | 18_000₽ | 23.9% | #######################
   food | 26_000₽ | 34.5% | ##################################
  train |  3_536₽ |  4.7% | ####
  phone |  8_556₽ | 11.4% | ###########
  music |  1_188₽ |  1.6% | #
parties | 18_000₽ | 23.9% | #######################

TOTAL: 75280₽
```

Логика построения таблицы:
1. Перебираем метрики и формируем список строк таблицы. Параллельно ищем ширину каждого столбца.
2. Перебираем строки и обновляем ячейки, выравнивая их в соответствие с шириной столбцов.
3. Отрисовываем строку за строкой.
